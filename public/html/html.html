Estructura

Etiquetas de Estructura HTML

HTML

html Define la apertura de el documento HTML

/html Es la etiqueta que da cierre al documento HTML

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                      <head>
                                        <!-- contenido del head -->
                                      </head>
                                      <body>
                                        <!-- contenido del body -->
                                      </body>
                                    </html>

HEAD

head Define la apertura del área del HTML donde almacenaremos metadatas y recurso a utilizar (Enlaces a estilos CSS, ETC)

/head Es la etiqueta que da cierre del segmento de metadatos y recursos

Importante: Esta etiqueta se utiliza antes del body, y es clave no confundirla con la etiqueta header

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                      <head>
                                      <meta charset="UTF-8">
                                      <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                      <title> Titulo de pestaña del documento </title>                                    


                                      <!-- Esta información se genera automaticamente al crear el html:5 -->
                                      </head>
                                      <body>
                                        <!-- contenido del body -->
                                      </body>
                                    </html>

BODY

body Define la apertura del cuerpo del documento HTML, siendo la zona donde se cargaran las etiquetas que formaran nuestra pagina.

/body Es la etiqueta que da cierre al segmento del cuerpo de nuestro documento

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                      <head>
                                        <!-- contenido del head -->
                                      </head>
                                      <body>
                                        <!-- Contenido del body -->
                                        <!-- Es la sección donde se definen todas las etiquetas a utilizar en nuestro sitio. -->
                                        <!-- Otra forma de entenderlo, es como el contenedor principal del documento (contiene todo).  -->
                                      </body>
                                    </html>

TITLE

title Define la apertura de la etiqueta encargada de definir el titulo de la pestaña de la pagina.

/title Es la etiqueta que da cierre a este apartado del nombre de la pestaña 

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                      <head>
                                      <title> Titulo de pestaña del documento </title>                                    
                                      <!-- Lo ideal es que tenga una referencia a la ubicación dentro del sitio web -->
                                      <!-- Por ejemplo: Esta pagina donde estas, su title es BlogTech|HTML -->
                                      </head>
                                      <body>
                                      </body>
                                    </html>

META

meta En HTML se usa esta etiqueta para definir metadatos sobre la página web

Esta etiqueta hace parte de las vacias, por lo cual no lleva una etiqueta de cierre.

Tip: Su localización es dentro del head, pues esta no brinda información visible, sino datos ocultos para el adecuado funcionamiento en navegadores y motores de busqueda.

                                
Ejemplo 

                                
                                      <meta charset="UTF-8">
                                      <!-- Esto asegura que los caracteres especiales (como tildes y ñ) se muestren correctamente. -->


                                      <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                      <!-- Permite que la web se adapte correctamente a dispositivos móviles (Responsive Site). -->

A

a es la etiqueta que indica el inicio del enlace de navegación a una Url especifica/p>

/a es la etiqueta que indica el final del enlace 

                                
Ejemplo 

                                
                                        href
                                        <!-- es el atributo que especifica la URL a la que se enlaza --> 
                                        "https://ejemplo.com"
                                        <!-- Es la URL a la que se enlaza -->

                                        <a href="https://ejemplo.com"></a>
                                        <!-- Los enlaces en HTML también se conocen como hipervínculos. -->

LINK

link se usa principalmente para enlazar archivos externos, como hojas de estilo CSS o íconos

Esta etiqueta hace parte de las vacias, por lo cual no lleva una etiqueta de cierre.

                                
Ejemplo 

                                
                                        <link rel="icon" type="image/png" href="favicon.png">
                                        <!-- Agregar un ícono a la pestaña del navegador (favicon). --> 
                                        <link rel="stylesheet" href="styles.css">
                                        <!-- Nos permite enlazar una hoja de estilos CSS.-->
                                        <link rel="preconnect" href="https://fonts.googleapis.com">
                                        <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap">>
                                        <!-- Se utilizan para Google Fonts. -->

STYLE

La etiqueta style e usa para definir estilos CSS dentro del mismo documento. Se coloca dentro del head y permite escribir reglas CSS que afectan a los elementos de la página

Esta etiqueta tambien se puede usar en el body, pero no es recomendable

                                        
Ejemplo 

                                        
                                            <!doctype html>
                                            <html lang="en">
                                              <head>
                                              <meta charset="UTF-8">
                                              <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                              <title> Titulo de pestaña del documento </title>                                    
                                              
                                              <style>
                                              
h1{

                                              
color: 🟥red;
  
                                              <!-- Aquí aplicamos una propiedad de color, donde los h1 se volveran de color rojo en todo el documento -->
                                              
  }

                                              </style>
    
                                              </head>
                                              <body>
    
                                              <h1>
Hola Mundo Tech
</h1>
    
                                              </body>
                                            </html>

SCRIPT

La etiqueta script se usa para incluir y ejecutar código JavaScript en una página web. Puede ubicarse dentro del head o al final del body, dependiendo de su uso.

Se recomienda colocar los scripts al final del body para mejorar el rendimiento de carga de la página.

                                    
Ejemplo 1: Script en Head 

                                        
                                            <!doctype html>
                                            <html lang="en">
                                              <head>
                                              <meta charset="UTF-8">
                                              <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                              <title> Ejemplo de script con JS </title>                                    
                                              <script src ="index.js"></script>
                                              </head>
                                              
<body>
                                              </body>
                                            </html>
                                        
                                
                                    
Ejemplo 2: Script en Body 

                                        
                                            <!doctype html>
                                            <html lang="en">
                                              <head>
                                              <meta charset="UTF-8">
                                              <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                              <title> Ejemplo de script con JS </title>                                    
                                              </head>
                                              <body>
                                              <h1>
Hola Mundo Tech
</h1>
                                              <script>                                           
                                              <!-- Aqui se colocarán todas las funciones de JavaScript -->
                                              
</script>
                                              </body>
                                            </html>

DIV

La etiqueta div se usa como un contenedor genérico en HTML. Su finalidad es la de agrupar elementos y aplicar estilos o scripts.

Se emplea comúnmente para estructurar el diseño de una página y trabajar con CSS y JavaScript.

                                
Ejemplo: 

                                    
                                        <div>
                                        <h1>Hola Mundo Tech</h1>
                                        <p>Era gol de yepes</p>
                                        <p>Viva Colombia, viva falcaooooooooo</p>
                                        </div>
                                        <!-- E Agrupa un título y uno o varios párrafos dentro de un contenedor.-->

SPAN

La etiqueta span se utiliza para aplicar estilos o manipular partes específicas de un texto sin alterar la estructura del documento.

Es útil para resaltar palabras o aplicar efectos con CSS y JavaScript.

                                
Ejemplo: 

                                    
                                        <div>
                                        <h1>Hola Mundo <span style="color:🟦blue">Tech</span></h1>
                                        <!-- El color azul será aplicado unicamente a la palabra "Tech", la cual esta dentro del span -->
                                        </div>

HEADER

La etiqueta header en HTML se usa para definir la cabecera de una página o una sección dentro de ella. Generalmente, contiene el logo, el título, la barra de navegación o información introductoria.

  
                                
Ejemplo:

                                
                                    <header>
                                    
<h1>Mi sitio WEB </h1>
    <p>Bienvenido a mi página </p>
                                    
</header>
                                    
<!-- Aquí contiene el título y una pequeña descripción-->
                                
                                
                                    
Ejemplo 2: con estilos CSS

                                    
                                          <style>
                                          
header{

                                            
background-color: #333;

                                            
color: white;

                                            
padding: 20px;

                                            
text-align: center;

                                            
  }

                                            <style>
                                            <header>
                                            
<h1>Mi sitio WEB </h1>
                                            
</header>
                                            
<!-- Esto le da un fondo oscuro y texto blanco al header.-->

FOOTER

La etiqueta footer se usa para definir el pie de página de una página web o de una sección. Normalmente contiene información de contacto, derechos de autor, enlaces a redes sociales o menús secundarios.

                                
Ejemplo: 

                                    
                                        <footer>
                                        
<p>© 2025 Mi Sitio Web. Todos los derechos reservados.</p>
                                        
</footer>
                                        <!-- Muestra un mensaje de derechos de autor. -->
                                    
                            
                                
Ejemplo 2: con enlaces y redes sociales 

                                    
                                        <footer>
                                        <p>© 2025 Mi Sitio Web.</p>
                                        <nav>
                                        <a href="https://ejemplo.com"></a>
                                        </nav>
                                        </footer>
                                        <!-- Agrega enlaces a redes sociales y se abren en una nueva pestaña. -->

SECTION

La etiqueta section en HTML se usa para dividir el contenido en secciones temáticas dentro de una página web. Es útil para organizar mejor la información y mejorar la accesibilidad.

                                
Ejemplo: 

                                    
                                        <section id="servicios">
                                        
<h2>Sobre nosotros</h2>
                                        <p>Somos una empresa dedicada al desarrollo de nuevas tecnologias.</p>
                                        
</section>
                                        <!-- Agrupa un título y un párrafo dentro de una sección. Este la primera seccion -->

                                        
<section id="contacto">
                                        
<h2>Sobre nosotros</h2>
                                        <p>Somos una empresa dedicada al desarrollo de nuevas tecnologias.</p>
                                        
</section>
                                        <!-- Agrupa un título y un párrafo dentro de una sección. Esta es la segunda sección -->

ARTICLE

La etiqueta article en HTML se usa para representar contenido independiente y autocontenido, como noticias, publicaciones de blog, comentarios o productos.

/article Esta es la etiqueta de cierre.

   
                                
Ejemplo: 

                                    
                                        <article>
                                        
<h1>Título del artículo</h1>
                                        <p>Este es el contenido de un artículo independiente.</p>
                                        
</article>
                                        <!-- Envuelve un título y un párrafo que forman un contenido independiente. 
También se puede Se pueden crear varios article en la misma página. -->
                                    
                            
   
                                
Ejemplo con imagen y autor: 

                                    
                                        <article>
                                        
<h2>Como aprender HTML</h2>
                                        <img src="imagen.jpg" alt="imagen de HTML">
                                        <p>Publicado por BlofTech en 2025.</p>
                                        <p>HTML es el lenguaje base para crear páginas web.</p>
                                        
</article>
                                        <!-- Incluye una imagen, un autor y una fecha de publicación. -->

ASIDE

La etiqueta aside en HTML se usa para definir contenido relacionado o complementario al contenido principal. Generalmente se usa para barras laterales, anuncios, enlaces adicionales o información secundaria.

                            
Ejemplo: 

                            
                                <article>
                                
<h1>Título del artículo</h1>
                                <p>Este es el contenido de un artículo principal.</p>
                                
</article>
                                
<aside>
                                
<h2>Contenido relacionado con del artículo</h2>
                                <p>Este es un texto adicional que complementa el contenido
     del articulo principal.</p>
                                
</aside>
                                <!-- Se usa para mostrar otras noticias relacionadas con el artículo principal. -->

NAV

La etiqueta nav se usa para definir una sección de navegación que contiene enlaces a otras partes de la página o a otras páginas.

                                
Ejemplo: 

                                
                                    <nav>
                                    
<a href="index.html">Inicio</a>  |
                                    <a href="nosotros.html">Sobre nosotros</a>  |
                                    <a href="contacto.html">Contactanos</a>
                                    
</nav>
                                    <!-- Crea un menú de navegación simple con enlaces separados por una barra |. -->
                                
                            
                                
Ejemplo con una lista para mejorar la estructura: 

                                
                                    <nav>
                                    <ul>
                                    
<li><a href="index.html">Inicio</a></li>
                                    <li><a href="nosotros.html">Sobre Nosotros</a></li>
                                    <li><a href="contacto.html">Contatanos</a></li>
                                    
</ul>
                                    </nav>
                                    <!-- Usa una lista ul para hacer el menú más accesible y organizado. -->

nav se utiliza para definir una sección de navegación dentro de una página web.

Nota importante: Generalmente, contiene enlaces a otras partes del sitio o a páginas externas.

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                    <head>
                                    <meta charset="UTF-8">
                                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                    </head>
                                    <body>
                                    <nav>
                                        <ul>
                                            <li>
                                                    <a href="index.html">Inicio</a>
                                            </li>
                                            <li>
                                                <a href="nosotros.html">Nosotros</a>
                                            </li>
                                        </ul>
                                    </nav>
                                    </body>
                                    </html>

Texto

P

p Es la etiqueta que nos permite insertar parrafos a nuestra pagina, y se cierra con su etiqueta de cierre /p

H1 - H6

h1 Es la etiqueta que nos permite insertar un encabezado principal en nuestra página. Se usa para los títulos más importantes y se cierra con su etiqueta de cierre /h1

h2 Es la etiqueta que representa un subtítulo o un encabezado secundario, con menor importancia que h1. Se cierra con su etiqueta de cierre /h2

h3 Es la etiqueta utilizada para encabezados de tercer nivel, generalmente para secciones dentro de un tema más amplio. Se cierra con su etiqueta de cierre /h3

h4 Es la etiqueta usada para títulos de menor relevancia dentro de un documento, como subtítulos dentro de una sección. Se cierra con su etiqueta de cierre /h4

h5 Es la etiqueta para encabezados menos importantes, usados en estructuras más detalladas o anidaciones profundas. Se cierra con su etiqueta de cierre /h5

h6 Es la etiqueta para el encabezado con menor jerarquía, generalmente usada para notas, referencias o secciones muy específicas. Se cierra con su etiqueta de cierre /h6

                                
Ejemplo: 

                                    
                                        Este es un texto con tamaño H1
                                        Este es un texto con tamaño H2
                                        Este es un texto con tamaño H3
                                        Este es un texto con tamaño H4
                                        Este es un texto con tamaño H5
                                        Este es un texto con tamaño H6

STRONG

strong Es la etiqueta que nos permite resaltar texto con **énfasis fuerte** en nuestra página. Su contenido se muestra en **negrita** de manera predeterminada y se cierra con su etiqueta de cierre /strong

                                
Ejemplo: 

                                
                                <p> Esto es un <strong>mensaje importante.<strong> </p>
                                
                            
                                
Resultado en pantalla: 

                                                               
                                
Esto es un mensaje importante.

EM

em Es la etiqueta que nos permite enfatizar texto en nuestra página. Su contenido se muestra en cursiva y se cierra con su etiqueta de cierre /em

Este es un mensaje enfatizado en la oración.

                                
Ejemplo: 

                                
                                <p> Esto es un <em>texto importante en cursiva.<em> </p>
                                
                            
                                
Resultado en pantalla: 

                                                               
                                
Esto es un texto importante en cursiva.

BR

br Es la etiqueta que nos permite insertar un salto de línea en nuestra página web. No necesita etiqueta de cierre porque es una etiqueta auto-cerrada.

                                
Ejemplo: 

                                
                                <p> Esto es un texto en primera linea. <br>Y este un texto en segunda linea.</p>
                                
                            
                                
Resultado en pantalla: 

                                                               
                                
Este es un texto en la primera línea.
Y este es un texto en la segunda línea.

HR

hr Es la etiqueta que nos permite insertar una línea horizontal en nuestra página web. Se usa para separar secciones de contenido y no necesita etiqueta de cierre porque es una etiqueta auto-cerrada.
