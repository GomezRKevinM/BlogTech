Etiquetas de Estructura HTML

HTML

html Define la apertura de el documento HTML

/html Es la etiqueta que da cierre al documento HTML

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                      <head>
                                        <!-- contenido del head -->
                                      </head>
                                      <body>
                                        <!-- contenido del body -->
                                      </body>
                                    </html>

HEAD

head Define la apertura del área del HTML donde almacenaremos metadatas y recurso a utilizar (Enlaces a estilos CSS, ETC)

/head Es la etiqueta que da cierre del segmento de metadatos y recursos

Importante: Esta etiqueta se utiliza antes del body, y es clave no confundirla con la etiqueta header

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                      <head>
                                      <meta charset="UTF-8">
                                      <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                      <title> Titulo de pestaña del documento </title>                                    


                                      <!-- Esta información se genera automaticamente al crear el html:5 -->
                                      </head>
                                      <body>
                                        <!-- contenido del body -->
                                      </body>
                                    </html>

BODY

body Define la apertura del cuerpo del documento HTML, siendo la zona donde se cargaran las etiquetas que formaran nuestra pagina.

/body Es la etiqueta que da cierre al segmento del cuerpo de nuestro documento

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                      <head>
                                        <!-- contenido del head -->
                                      </head>
                                      <body>
                                        <!-- Contenido del body -->
                                        <!-- Es la sección donde se definen todas las etiquetas a utilizar en nuestro sitio. -->
                                        <!-- Otra forma de entenderlo, es como el contenedor principal del documento (contiene todo).  -->
                                      </body>
                                    </html>

TITLE

title Define la apertura de la etiqueta encargada de definir el titulo de la pestaña de la pagina.

/title Es la etiqueta que da cierre a este apartado del nombre de la pestaña 

                                
Ejemplo 

                                
                                    <!doctype html>
                                    <html lang="en">
                                      <head>
                                      <title> Titulo de pestaña del documento </title>                                    
                                      <!-- Lo ideal es que tenga una referencia a la ubicación dentro del sitio web -->
                                      <!-- Por ejemplo: Esta pagina donde estas, su title es BlogTech|HTML -->
                                      </head>
                                      <body>
                                      </body>
                                    </html>

META

meta En HTML se usa esta etiqueta para definir metadatos sobre la página web

Esta etiqueta hace parte de las vacias, por lo cual no lleva una etiqueta de cierre.

Tip: Su localización es dentro del head, pues esta no brinda información visible, sino datos ocultos para el adecuado funcionamiento en navegadores y motores de busqueda.

                                
Ejemplo 

                                
                                      <meta charset="UTF-8">
                                      <!-- Esto asegura que los caracteres especiales (como tildes y ñ) se muestren correctamente. -->


                                      <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                      <!-- Permite que la web se adapte correctamente a dispositivos móviles (Responsive Site). -->

A

a es la etiqueta que indica el inicio del enlace de navegación a una Url especifica/p>

/a es la etiqueta que indica el final del enlace 

                                
Ejemplo 

                                
                                        href
                                        <!-- es el atributo que especifica la URL a la que se enlaza --> 
                                        "https://ejemplo.com"
                                        <!-- Es la URL a la que se enlaza -->

                                        <a href="https://ejemplo.com"></a>
                                        <!-- Los enlaces en HTML también se conocen como hipervínculos. -->

LINK

link se usa principalmente para enlazar archivos externos, como hojas de estilo CSS o íconos

Esta etiqueta hace parte de las vacias, por lo cual no lleva una etiqueta de cierre.

                                
Ejemplo 

                                
                                        <link rel="icon" type="image/png" href="favicon.png">
                                        <!-- Agregar un ícono a la pestaña del navegador (favicon). --> 
                                        <link rel="stylesheet" href="styles.css">
                                        <!-- Nos permite enlazar una hoja de estilos CSS.-->
                                        <link rel="preconnect" href="https://fonts.googleapis.com">
                                        <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap">>
                                        <!-- Se utilizan para Google Fonts. -->
