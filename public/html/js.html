    • ¿Qué es Javascript?

    JavaScript(JS) es un lenguaje de programación ligero, interpretado, o compilado justo-a-tiempo (just-in-time) con funciones de primera clase. Si bien es más conocido como un lenguaje de scripting (secuencias de comandos) para páginas web, y es usado en muchos entornos fuera del navegador, tal como Node.js, Apache CouchDB y Adobe Acrobat JavaScript es un lenguaje de programación basada en prototipos, multiparadigma, de un solo hilo, dinámico, con soporte para programación orientada a objetos, imperativa y declarativa (por ejemplo programación funcional).

    Variables

    Una variable es una referencia guarda en la ram, esta referencia es capaz de almacenar información para luego solicitarla e implementarla en algun tipo de algoritmo, función, etc. En otras palabras una variable puede contener información almacenada y darnos esta información cuando la necesitemos 

                            
    Ejemplo 
    
                        
                             const mi_nombre = "carlos";
                             console.log(mi_nombre); //carlos

    Tipos de variables

    • const

      const es un tipo de variable constante. Es decir que su valor es inmutable, no puede ser cambiado. ejemplos de variables constantes: la gravedad,id,PI,etc.

    • let

      let es un tipo de variable general, esta variable puede almacenar diferentes valores y tiene permitido cambiar su valor en un futuro (es decir, puedes cambiar su valor tantas veces como quieras o lo necesites)

      Ejemplos de variables let: la edad, estatura, velocidad,etc.
    • var

      var al igual que let es una variable general, funciona exactamente igual a let pero var es la precursora de let (es decir, es más vieja). pero, var suele presentar problemas en tiempos de ejecución al ser una variable casi obsoleta. Todo el mundo del desarrollo web apunta e incita a usar let y dejar var en el pasado (sin embargo puedes usarla aún si quieres pero recuerda que es recomendable usar let)

    Tipos de datos

    • Cadena de texto (String)
    • Number (int, float, double)
    • Booleanos (true,false,comparaciones)
    • Vacias o indefinidas (null, undefine, NaN)

    Las variables deben ser declaradas para posteriormente recibir una asignación(Asignarle un valor), ¿como podemos hacer esto?
    Primero comencemos por comprender que es una Declaración y una Asignación
    Declaración: La Declaración es el termino que recibe una variable al ser declarada(es decir, creada).
    Asiganción: La Asignación es el termino que recibe la acción de asignarle (darle o guardar) un valor a la variable.
    IMPORTANTE:Con la anterior información, se puede deducir que no se puede asignar un valor a una variable sin haberla declarado antes, puesto que si no ha sido declarada simplemente no existe y por ende no puede almacenar un valor.

    Como declarar una variable

    Para declarar una variable, debemos escribr primero que tipo de variable es (const, let o var) y luego darle un nombre. 

                                
    Ejemplo
    
                            
                                const id; // id se encuentra declarada

    Como asignar un valor a una variable

    Para asignar un valor a una variable empleamos el signo de igualdad "=", esto lo podemos hacer ya sea cuando declaramos la variable o posteriormente al llamar a la variable. 

                            
    Asignar en la misma declaración
    
                        
                            const cedula  = 1001974052;// aqui declaramos (creamos la variable) y al mismo tiempo le asignamos un valor.
                        
                    
                            
    Asignar valor llamando a la variable
    
                        
                            const edad;// declaramos la variable strong edad
    
                            const answer = prompt("¿Cual es tu edad?"); // pedimos información al usuario usando un prompt (mas adelante veremos qué es). 
    
                            edad = answer;// le asignamos el valor que nos ingreso el usuario.

    Condicionales

    Los condicionales es la forma de otorgar condiciones a nuestro código, dandole opciones de ejecución segun una o multiples condiciones.

    Condicionales Basicos

    • if

      IF(sí), es el encargado de recibir la condición inicial, al igual que en nuestro español sí hace referencia a un resultado depentiende. Ejemplo:yo comprare ropa sí me pagan.

    • else if

      ELSE IF comparte las mismas propiedades que IF, la unica diferencia es que if es la condición principal(la primera en ser leida y analizada por el sistema) mientras que else if solo participa si la condición de if no es cumplida. (como una condición adicional en caso de que no se cumpla la primera).
      Puedes crear tantas else if como necesites.

    • else

      ELSE es la negación de todas las condiciones anteriores, es decir, en else se almacena el código que sera ejecuta en cualquier caso que no se cumpla la o las condiciones anteriores (if y else if)
      Else no necesita una condición ya que su condición es que no se cumpla ningun if o else if.

    Estructuras

                            
    Estructura de un if
    
                        
                            if (condición){
                                // código aquí
                                }
                        
                    
                            
    Estructura de un else if
    
                        
                            else if (condición){
                                 // código aquí
                            }
                        
                    
                            
    Estrucutra de un else
    
                        
                            else{
                            // código aquí
                            }

    Condicionales avanzandos

    Los condicionales avanzados son switch case y El operador ternario, teniendo usos practicos y especificos para ocasiones especiales.

    • Switch:

      Switch case se utiliza para emplear condiciones espcificas llamadas case (caso) en relación a una variable.
      El switch utiliza los casos para evaluar valores o resultados puntuales de una variable.
      IMPORTANTE: switch es util para aquellos casos donde queremos definir comportamientos o procesos a resultados especificos, y tambien para cuando tienes más de 3 condiciones, de esta manera el código se verá más organizado en case que en else if. 

                                  
      Ejemplo de switch
      
                            
                                let langSelect = "es";
                                 
                                 switch(langSelect){
                                  case "es":
                                      console.log("el usuario selecciono español")
                                  break;
                                  case "en":
                                      console.log("the user is select english")
                                  break;
                                  default:
                                // el default funciona como un else, es decir se ejecutará sí no se cumple ningun caso.
                                  break;
                                }
    • Operador ternario:

      El operador ternario nos permite hacer una condición de una sola linea: es decir, condición, resultado si la condición es verdadera (sí se cumple) y resultado si la condición es falsa (no se cumple)
      IMPORTANTE: El operador ternario es util solo y cuando tengas 1 condición y quieras establecer 2 comportamientos distintos para esa condición ( 1 comportamiento sí se cumple y otro sí no se cumple). 

                                  
      Ejemplo de Operador ternario
      
                            
                                let edad = 14;
                                let esMayor = edad >= 18 ? "Es mayor de edad" : "No es mayor de edad";
                                consolelog(esMayor) // No es mayor de edad

    Bucles

    Los bucles son estructuras de control que permiten repetir un bloque de código varias veces. Son útiles cuando necesitas ejecutar el mismo código múltiples veces con diferentes valores. Los bucles más comunes en JavaScript son for, while y do...while.

    Tipos de Bucles

    • For

      El bucle for se utiliza cuando conoces de antemano el número de iteraciones que deseas realizar. La estructura básica de un bucle for incluye una inicialización (i = 0 ), una condición (i < 5) y una expresión (i++) final.
      El Bucle inicia con la variable de inicialización, posteriormente evalua la condición, sí esta se cumple entonces ejecuta el bloque y luego ejecuta la expresión para incrementar o decrementar la variable de inicialización y repetir el mismo ciclo. 

                                  
      Ejemplo de for
      
                            
                                for (let i = 0; i < 5; i++) {
                                    console.log(i);
                                }
                            
                        
                                  
      Resultado
      
                            
                                 0
                                 1
                                 2
                                 3
                                 4
    • While

      El bucle while se utiliza cuando deseas repetir un bloque de código mientras una condición sea verdadera. La condición se evalúa antes de cada iteración del bucle. Mientras(while) la condición se cumpla, este se ejecutará

                                  
      Ejemplo de while
      
                            
                                let i = 0;
                                while (i < 5) {
                                    console.log(i);
                                    i++;
                                }
                            
                        
                                  
      Resultado
      
                            
                                 0
                                 1
                                 2
                                 3
                                 4
    • Do-While

      El bucle do...while es similar al bucle while, pero garantiza que el bloque de código se ejecute al menos una vez antes de evaluar la condición, porque este primero ejecuta el do y luego evalua la condición para saber si puede iterar. Es decir, si la primera condición no se cumple, el do-while se habra ejecutado una vez antes de comprobar que esta condición no se cumple.

                                  
      Ejemplo de do-while
      
                            
                                let i = 0;
                                do {
                                    console.log(i);
                                    i++;
                                }while (i < 5);
                            
                        
                                  
      Resultado
      
                            
                                 0
                                 1
                                 2
                                 3
                                 4
    • ForEach

      El método forEach ejecuta una función proporcionada una vez por cada elemento del array (es decir, hace una iteración por cada elemento existente en el array ). Es útil para iterar sobre los elementos de un array sin necesidad de un contador explícito.

                                  
      Ejemplo de forEach
      
                            
                                const array = [1, 2, 3, 4, 5];
                                array.forEach((element) {
                                    console.log(element);
                                });
                            
                        
                                  
      Resultado
      
                            
                                 1
                                 2
                                 3
                                 4
                                 5
