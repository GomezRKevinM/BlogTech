¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en HTML o XML. CSS describe cómo los elementos HTML deben ser renderizados en pantalla, en papel, en voz, o en otros medios.

Selectores en CSS

Los selectores en CSS se utilizan para seleccionar los elementos HTML que deseas estilizar. Hay varios tipos de selectores:

  • Selector de etiqueta: Selecciona todos los elementos de un tipo específico. Ejemplo: p { color: red; }
  • Selector de clase: Selecciona todos los elementos con una clase específica. Ejemplo: .miClase { color: blue; }
  • Selector de ID: Selecciona un elemento con un ID específico. Ejemplo: #miID { color: green; }
  • Selector de atributo: Selecciona elementos con un atributo específico. Ejemplo: [type="text"] { color: purple; }
  • Selector de descendiente: Selecciona elementos que son descendientes de otro elemento. Ejemplo: div p { color: orange; }
  • Selector de hijo: Selecciona elementos que son hijos directos de otro elemento. Ejemplo: div > p { color: pink; }
  • Selector de hermano adyacente: Selecciona un elemento que es el siguiente hermano inmediato de otro elemento. Ejemplo: h1 + p { color: brown; }
  • Selector de hermano general: Selecciona todos los elementos hermanos de un elemento. Ejemplo: h1 ~ p { color: gray; }

Especificidad en CSS

La especificidad determina qué reglas CSS se aplican a un elemento cuando hay múltiples reglas que podrían aplicarse. La especificidad se calcula en base a cuatro valores: inline styles, IDs, clases y atributos, y elementos.

  • Estilos en línea: Tienen la mayor especificidad. Ejemplo: <h1 style="color: red;">
  • IDs: Tienen una especificidad alta. Ejemplo: #miID { color: green; }
  • Clases y atributos: Tienen una especificidad media. Ejemplo: .miClase { color: blue; }
  • Elementos: Tienen la menor especificidad. Ejemplo: p { color: red; }

Propiedades CSS

CSS tiene una amplia variedad de propiedades que se pueden aplicar a los elementos HTML. Aquí hay algunas propiedades básicas y avanzadas:

Propiedades Básicas

  • color: Define el color del texto. Ejemplo: color: red;
  • background-color: Define el color de fondo de un elemento. Ejemplo: background-color: yellow;
  • font-size: Define el tamaño de la fuente. Ejemplo: font-size: 16px;
  • margin: Define el margen exterior de un elemento. Ejemplo: margin: 10px;
  • padding: Define el relleno interior de un elemento. Ejemplo: padding: 10px;
  • border: Define el borde de un elemento. Ejemplo: border: 1px solid black;

Propiedades Avanzadas

  • box-shadow: Añade sombra a un elemento. Ejemplo: box-shadow: 2px 2px 5px gray;
  • transition: Define la transición entre dos estados de un elemento. Ejemplo: transition: all 0.3s ease;
  • transform: Aplica una transformación a un elemento. Ejemplo: transform: rotate(45deg);
  • flex: Define cómo un elemento flexible crecerá o se encogerá. Ejemplo: flex: 1;
  • grid: Define un contenedor de cuadrícula. Ejemplo: display: grid;

Media Queries

Las media queries se utilizan para aplicar estilos CSS específicos en función de las características del dispositivo, como el ancho de la pantalla. Esto es esencial para el diseño responsive. 

                
                    @media (max-width: 600px) {
                        body {
                            background-color: lightblue;
                        }
                    }

Keyframes para Animaciones

Los keyframes se utilizan para definir una secuencia de estilos CSS que crean una animación. Puedes especificar los estados inicial y final, así como cualquier punto intermedio. 

                
                    @keyframes example {
                        0% { background-color: red; }
                        50% { background-color: yellow; }
                        100% { background-color: green; }
                    }

                    .animated-element {
                        animation: example 5s infinite;
                    }

Pseudo-elementos

Los pseudo-elementos se utilizan para aplicar estilos a partes específicas de un elemento. Los más comunes son ::before y ::after. 

                
                    p::before {
                        content: "Inicio: ";
                        color: red;
                    }

                    p::after {
                        content: " Fin.";
                        color: blue;
                    }
